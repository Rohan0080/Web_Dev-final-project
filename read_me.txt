Web dev final project

This document contains a complete starter pack for your MIT507 group assignment: file structure, HTML/CSS/JS/PHP code samples, MySQL schema, and a report template. Copy the files into your website folder and test on a local PHP+MySQL server (e.g., XAMPP, MAMP).

---

## Project file structure

```
WEB_DEV FINAL PROJECT/
├─ index.html                # Home (consultantsite.html)
├─ solutions.php             # Solutions page (reads from array or DB)
├─ services.php              # Services listing
├─ request_demo.php          # Form and processing (POST to submit_demo.php)
├─ submit_demo.php           # Server-side processing + DB insert
├─ about.html                # About Us
├─ css/
│  └─ styles.css
├─ js/
│  └─ script.js
├─ images/
│  └─ logo.png
├─ inc/
│  ├─ config.php            # DB credentials (include in .gitignore)
│  └─ helpers.php           # helper functions (validation)
└─ sql/
   └─ schema.sql            # SQL to create DB and table
```
